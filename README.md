## 🧠 Módulos Python

### 🔹 `num_primos.py`

Funções para trabalhar com números primos:

- `eh_primo(n)` → verifica se número é primo
- `listar_primos(ate)` → lista primos até um limite

✔ Otimizado com:
- verificação até √n
- eliminação de pares

---

### 🔹 `refatoracao.py`

Cálculo de estatísticas:

- Total
- Média
- Maior valor
- Menor valor

Uso de boas práticas:
- `NamedTuple`
- Type hints
- Funções built-in (`sum`, `max`, `min`)

---

### 🔹 `debug.py`

Simulação de um sistema de compras com:

- Entrada de dados do usuário
- Cálculo de subtotal
- Aplicação de imposto (10%)
- Aplicação de desconto via cupom
- Exibição formatada

Ideal para treino de:
- Debug
- Tipagem
- Lógica de negócio
